# Accuretta · Mobile UI Kit

The companion mobile surface. Primary jobs-to-be-done: monitor running sessions, approve/review plans, read transcripts, view run history.

## Files
- `index.html` — click-through demo. Tap the bottom nav to switch screens.
- `components.jsx` — `StatusBar`, `AppBar`, `BottomNav`, `SessionsScreen`, `PlanScreen`, `RunsScreen`, `SettingsScreen`, `SessionRow`, `MIcon`.
- `styles.css` — all `mb-*` classes.

## Screens
1. **Sessions** — searchable list of active + past sessions.
2. **Plan** — live running step + queue + approve-remaining CTA.
3. **Runs** — last 7 days stats + recent run log.
4. **Settings** — profile, models, preferences (dark mode, local-only, notifications).
